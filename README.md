# Quiz Application (Core Java)

A desktop-based Quiz Application developed using **Core Java**, **Swing**, and **AWT**.  
The application allows users to attempt multiple-choice questions and view their final score.

## Features
- Desktop GUI using Java Swing and AWT
- Multiple-choice questions
- One question displayed at a time
- Score calculation at the end of the quiz
- Simple and user-friendly interface

## Technologies Used
- Core Java
- Java Swing
- AWT
- OOP Concepts (Encapsulation, Inheritance, Polymorphism)

## Core Java Concepts Used
- Classes and Objects
- Inheritance
- Interfaces
- Event Handling
- Exception Handling
- Collections (if applicable)

## Application Flow
1. User starts the quiz
2. Questions are displayed one by one
3. User selects answers using radio buttons
4. Final score is displayed after quiz completion

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ)
3. Compile and run the main Java class
4. Quiz window will open on the desktop

## Purpose
This project demonstrates strong fundamentals of **Core Java and GUI development**, suitable for entry-level Java developer roles.
